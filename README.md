# VoiceCanvas AI

VoiceCanvas AI 是一个面向流程图、结构图和简单示意图创作的 AI 语音绘图 Web Demo。用户可以通过中文语音或文本指令创建图形、修改对象、连接对象、删除对象，并在指令存在歧义时收到明确提示。

一句话介绍：

VoiceCanvas AI 是一款通过自然语言语音指令控制画布的 AI 绘图工具，用户可以用语音创建图形、修改颜色和位置、连接对象、删除对象，并在存在歧义时由系统主动确认。

## 设计思路

项目采用“结构化画布状态 + 对象编号 + 本地规则解析 + 歧义提示”的方案。

画布中的每个元素都有唯一 ID，例如：

- 圆形 A
- 矩形 B
- 菱形 C
- 箭头 D

用户后续可以通过 ID 精确操作对象，例如“把矩形 B 改成绿色”。如果用户说“删除那个图形”这类无法明确定位对象的指令，系统不会直接执行，而是提示用户使用对象编号重新说明。

核心数据结构存储在 React state 中，SVG 根据 state 渲染画布，对象列表也读取同一份 state，保证画布和列表始终同步。

## 成本控制策略

- 使用浏览器 Web Speech API 完成语音识别，减少服务端语音识别成本。
- 简单指令优先使用本地规则解析，不调用大模型。
- 如后续接入大模型，仅将模型用途限制在“指令解析”，不做图像生成。
- 画布渲染使用 SVG 在前端完成，不产生额外计算成本。
- 保留结构化画布状态，减少重复解析和无效请求。
