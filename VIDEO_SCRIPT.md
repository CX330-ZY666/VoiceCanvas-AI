# VoiceCanvas AI 演示视频脚本

## 视频目标

建议视频控制在 3 到 5 分钟，声音讲解清楚，覆盖核心模块和主要效果。

## 录制前准备

1. 打开终端，运行：

```bash
npm install
npm run dev
```

2. 打开浏览器访问：

```text
http://localhost:3000
```

3. 确认麦克风权限已允许。

## 讲解开场

大家好，这是我的作品 VoiceCanvas AI，一款 AI 语音绘图 Web Demo。它主要用于通过中文语音指令创建和编辑流程图、结构图和简单示意图。项目使用 Next.js、React、TypeScript、Tailwind CSS、SVG 和浏览器 Web Speech API 实现，不依赖后端服务和 API Key。

## 演示步骤

### 1. 页面结构

讲解：

页面左侧是控制区，中间是 SVG 画布，右侧是对象列表。画布中的每个对象都会自动编号，例如 A、B、C，方便后续通过语音精确操作。

### 2. 基础创建

操作：

- 输入或说出“画一个红色圆形”
- 输入或说出“在圆形 A 右边画一个蓝色矩形”

讲解：

系统会把中文指令解析为结构化命令，并在 SVG 画布上创建对象。

### 3. 连接对象

操作：

- 输入或说出“连接 A 和 B”

讲解：

系统会创建箭头对象，连接两个图形。

### 4. 修改对象

操作：

- 输入或说出“把矩形 B 改成绿色”
- 输入或说出“把圆形 A 往右移动一点”
- 输入或说出“把 A 的文字改成开始”

讲解：

系统支持修改颜色、位置、大小和文字标签。

### 5. 歧义提示

操作：

- 输入或说出“删除那个图形”

讲解：

当指令目标不明确时，系统不会误删对象，而是提示用户使用对象编号重新说明。

### 6. 多轮上下文

操作：

- 输入或说出“选中 A”
- 输入或说出“把它改成绿色”

讲解：

系统会记住当前选中的对象，把“它”解析成对象 A。

### 7. 语音帮助和状态查询

操作：

- 说“我能说什么”
- 说“画布里有什么”
- 说“历史记录”

讲解：

系统支持语音帮助、画布摘要和最近命令历史，方便用户在纯语音场景中理解当前状态。

### 8. 安全确认

操作：

- 说“清空画布”
- 说“取消”

讲解：

清空画布是危险操作，所以系统要求二次确认，避免误操作。

### 9. 高级能力

操作：

- 说“复制 A”
- 说“整理画布”
- 说“导出画布”

讲解：

系统支持复制对象、一键整理布局和导出 SVG 文件。

### 10. 模板能力

操作：

- 输入或说出“生成一个登录流程图”

讲解：

系统可以通过一句话生成登录流程图模板，适合快速演示完整流程图能力。

## 结尾

讲解：
