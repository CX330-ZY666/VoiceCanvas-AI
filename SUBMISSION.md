# VoiceCanvas AI 提交说明

## 项目名称

VoiceCanvas AI

## 项目简介

VoiceCanvas AI 是一个通过中文语音或文本指令控制 SVG 画布的流程图 / 结构图 Web Demo。项目重点不是自由绘画，而是让用户用自然语言快速创建和编辑结构化示意图。

## 运行方式

```bash
npm install
npm run dev
```

访问：

```text
http://localhost:3000
```

## 提交材料清单

- GitHub 仓库：`https://github.com/CX330-ZY666/VoiceCanvas-AI`

- Demo 视频链接：录制并上传后填写到 README

## 推荐演示路径

1. 输入或说出“画一个红色圆形”。
2. 输入或说出“在圆形 A 右边画一个蓝色矩形”。
3. 输入或说出“连接 A 和 B”。
4. 输入或说出“把矩形 B 改成绿色”。
5. 输入或说出“删除那个图形”，展示歧义提示。
6. 输入或说出“选中 A”，再说“把它改成绿色”。
7. 输入或说出“画布里有什么”，展示画布摘要。
8. 输入或说出“清空画布”，展示二次确认。
9. 输入或说出“整理画布”，展示自动排版。
10. 输入或说出“导出画布”，展示 SVG 导出。
11. 输入或说出“生成一个登录流程图”，展示一键模板能力。

## 核心亮点

- 语音和文本共用同一套指令解析与执行链路。
- 普通绘图指令优先通过服务端大模型解析，支持 OpenAI Responses API 和 DeepSeek Chat Completions API，能处理更口语化的自然语言表达。
- 每个对象自动编号，降低后续语音操作歧义。
- SVG 前端渲染，结构清晰，成本低。
- 未配置 API Key 时自动回退本地规则解析，保证 Demo 仍可运行。
- 歧义指令不会误操作，会要求用户明确对象 ID。
- 登录流程图模板适合比赛现场快速展示。
- 支持语音反馈、语音帮助、画布摘要、命令历史、代词承接和导出 SVG。
