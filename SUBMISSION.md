# VoiceCanvas AI 提交说明

## 项目名称

VoiceCanvas AI

## 项目简介

VoiceCanvas AI 是一个通过中文语音或文本指令控制 SVG 画布的流程图 / 结构图 Web Demo。项目重点不是自由绘画，而是让用户用自然语言快速创建和编辑结构化示意图。

## 运行方式

```bash
npm install
npm run dev
```

访问：

```text
http://localhost:3000
```

## 推荐演示路径

1. 输入“画一个红色圆形”。
2. 输入“在圆形 A 右边画一个蓝色矩形”。
3. 输入“连接 A 和 B”。
4. 输入“把矩形 B 改成绿色”。
5. 输入“把圆形 A 往右移动一点”。
6. 输入“删除那个图形”，展示歧义提示。
7. 输入“撤销”，展示历史恢复。
8. 点击“开始语音输入”，说出“画一个黄色菱形”。
9. 输入“生成一个登录流程图”，展示一键模板能力。

## 核心亮点

- 语音和文本共用同一套指令解析与执行链路。
- 每个对象自动编号，降低后续语音操作歧义。
- SVG 前端渲染，结构清晰，成本低。
- 本地规则解析覆盖主要演示指令，无需 API Key 也能运行。
- 歧义指令不会误操作，会要求用户明确对象 ID。
- 登录流程图模板适合比赛现场快速展示。

## 验证命令

```bash
npm run lint
npm run build
```

## 开发说明

项目按 10 个独立 PR 逐步实现，避免最后一次性导入全部代码。每个 PR 保持单一目标，并保证合并后 `main` 分支可以继续运行 `npm run dev`。
